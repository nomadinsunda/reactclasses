## 1. Vite 프로젝트 생성
터미널(또는 CMD)을 열고 아래 명령어를 순서대로 입력하세요.

```bash
# 1. 프로젝트 생성 (vite-react-router-sample-extended 부분은 원하는 이름으로 변경 가능)
npm create vite@latest vite-react-router-sample-extended

# 2. 프로젝트 폴더로 이동
cd vite-react-router-sample-extended

# 3. 기본 의존성 설치
npm install
```

## 2. React Router 설치
라우팅을 위해 필요한 라이브러리를 추가로 설치합니다.

```bash
npm install react-router-dom
```
